**Action:** Runs the final tour fix prompt

## How to Add These to Cursor

1. Open Cursor Settings (Ctrl+,)
2. Go to "Custom Commands" or "AI Commands"
3. Add each command with its corresponding prompt:

**Fix tour:**
```
Fix the onboarding tour card positioning. It must be fixed at bottom: bottom-8 on desktop, bottom-20 on mobile. Ensure it never covers the mobile bottom nav.
```

**Kill Q:**
```
Find and delete any spinning Q logo immediately. Remove all traces of spinning animations on the Q logo. The spinning Q is banned forever.
```

**Add exchange:**
```
Ensure Kraken and Binance cards are present in API Integrations. If they are missing, add them immediately. These cards must always be present.
```

**Make solid pink:**
```
Convert the title to solid hot pink #FF1493. Remove any rainbow gradients, animations, or glow effects. It must be solid color only.
```
